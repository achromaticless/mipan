# My Pan — Panini Sticker Maker

A hobby open source web app to create your own Panini-style FIFA World Cup 26 sticker. Upload a photo, fill in your details, and download a print-ready PNG.

## What it does

- Upload a player photo
- Enter name, date, stats, and team label
- Live preview of the sticker card
- Download as a high-resolution PNG or a full A4 sticker sheet (8 per page)

## Stack

- React + Vite
- Pure HTML/CSS sticker rendering
- `html-to-image` for PNG export
- Bebas Neue / Saira fonts

## Getting started

```bash
npm install
npm run dev
```
